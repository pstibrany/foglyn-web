Getting Started

Welcome to Foglyn

With Foglyn, you will be able to work with your FogBugz cases from within Eclipse development environment or from applications based on Eclipse, like MyEclipse, Zend Studio, Aptana Studio, Adobe Flex Builder and many others.

Foglyn not only integrates FogBugz into your favorite IDE but it also enhances your interaction with cases:

  • thanks to Foglyn you can easily track cases from various subprojects or cases assigned to different members of your team
  • you can quickly navigate from the source code to the associated FogBugz case
  • thanks to integration with WikiText you can use extended formatting for your comments

This User Guide describes all parts of Foglyn. Since Foglyn is based on Mylyn, Task-focused interface for Eclipse, entire user guide will refer heavily to Mylyn Documentation. Foglyn User Guide also contains references to relevant documentation from FogBugz Online Help.
